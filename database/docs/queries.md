## Analytics questions that we will answer by writing queries...

There should these type of queries:
- KPI: returning a single number, like total no. of users, total videos etc Tabular: returns a set or records with joins and all that, which we can make a table from on a particular dashboard.
- Graphs: returns data suitable to make a illustrative graph.

Put queries appropriately in respective section, or introduce a new section if required.

### KPI

- Total no. of live users.
  - This will be the no. of users whose session is currently running.

### Tabular

- most viewed genre for a particular age range
  - filters: genre(select), age(select)
- time spent on a genre
  - filters: genre)(select), time(ascend/descend)
- age groups spending most time on a device for a particular range of time.

### Graphs

- Average length of sessions of users by age wise.
